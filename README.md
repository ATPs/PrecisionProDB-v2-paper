# PrecisionProDB-v2-paper
Scripts for reproduce the results in the PrecisonProDB v2 paper

# Source files

## CHM13-T2T resources

* chm13v2.0.fa were downloaded from https://github.com/marbl/CHM13 with [this link](https://s3-us-west-2.amazonaws.com/human-pangenomics/T2T/CHM13/assemblies/analysis_set/chm13v2.0.fa.gz).
* Liftover files were also downloaded from https://github.com/marbl/CHM13 page.
* GATK resources to call the variants were downloaded from [human-pangenomics](https://s3-us-west-2.amazonaws.com/human-pangenomics/index.html?prefix=T2T/CHM13/assemblies/variants/GATK_CHM13v2.0_Resource_Bundle/).
